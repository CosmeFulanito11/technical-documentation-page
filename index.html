Introducción

HTML5 es la última versión del lenguaje estándar para la creación de páginas web.

Fue desarrollado con el objetivo de mejorar la interoperabilidad entre navegadores, simplificar el desarrollo de aplicaciones web y brindar soporte nativo para elementos multimedia como audio, video y gráficos.

Esta versión representa una evolución significativa respecto a versiones anteriores, incorporando nuevas etiquetas semánticas, APIs potentes y un enfoque centrado en la accesibilidad y la experiencia del usuario.

Características principales

HTML5 introduce una variedad de características que modernizan el desarrollo web:

  • Etiquetas semánticas: como <header>, <section>, <article>, y <footer>, que mejoran la estructura y comprensión del contenido.
  • Soporte multimedia nativo: elementos como <audio> y <video> permiten integrar contenido sin plugins externos.
  • Formularios mejorados: nuevos tipos de entrada (email, date, range, etc.) y validación incorporada.
  • APIs integradas: como Web Storage, Canvas, WebSockets y Geolocation, que permiten funcionalidades avanzadas directamente en el navegador.
  • Compatibilidad móvil: pensado desde su base para adaptarse mejor a diferentes dispositivos y resoluciones.
Estructura básica de un documento HTML5

Un documento HTML5 se inicia con una declaración simple de tipo de documento y una estructura clara. A continuación, un ejemplo mínimo:

        
<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Título de la página</title>
  </head>
  <body>
    <header>
      <h1>Bienvenido a HTML5</h1>
    </header>
    <main>
      <section>
        <p>Contenido principal aquí.</p>
      </section>
    </main>
    <footer>
      <p>© 2025 Mi Sitio Web</p>
    </footer>
  </body>
</html>

Esta estructura básica incluye elementos clave como <!DOCTYPE html>, etiquetas semánticas y metadatos esenciales para la correcta visualización y accesibilidad.

Buenas prácticas

Aplicar buenas prácticas al escribir HTML5 mejora la mantenibilidad, accesibilidad y rendimiento del sitio web.

Algunas recomendaciones:

  • Usar etiquetas semánticas para mejorar la estructura del contenido.
  • Mantener una indentación coherente y un código limpio.
  • Agregar atributos alt en imágenes para accesibilidad.
  • Separar el contenido del estilo y comportamiento, utilizando CSS y JavaScript en archivos externos.
  • Validar el código HTML con herramientas como el validador de W3C.
